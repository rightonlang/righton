# Examples

## Hello World

```text
fn main():
    print("Hello, Righton!")
    return 0
```

## Variables and Arithmetic

```text
fn main():
    let a: i32 = 5
    let b: i32 = 3
    let c = a * b + 2
    print(c)
    return 0
```

## Strings and F-Strings

```text
fn main():
    let name = "World"
    print(f"Hello {name}")
    return 0
```
